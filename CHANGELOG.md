## [0.1.0] - 2025-12-18

### Added
- Initial public release of RoK Vision API
- Cognitive Orchestrator with specialized Neurons and semantic anchor-based data location
- The Magnifier: Auto-healing engine for low-confidence regions using cropping, filters, and parallel re-scanning
- High-performance OCR backend powered by PaddleOCR (PP-OCRv4) via Python FastAPI service
- Full governor profile extraction: ID, Name, Alliance (Tag & Name), Power, Kill Points, and Civilization
- Multilingual label recognition (EN, PT, CN, KR, RU, DE, FR, AR, and more)
- Smart image downscaling for 4K/2K screenshots with zero precision loss
- Fully containerized microservices architecture with Docker Compose support
- Hardware acceleration configuration (CUDA GPU / MKLDNN CPU)

### Changed
- Translated all code comments and logs to English for international open-source collaboration
